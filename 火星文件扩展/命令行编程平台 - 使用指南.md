# 🔴 火星编程 CODE - 使用指南

## 📖 产品简介

**火星编程 CODE** 是下一代智能编程平台，集成：
- 🤖 AI 智能对话（支持 8 种大模型）
- 🔴 火星文件管理
- 🔌 MCP 协议支持
- 🎯 精确代码编辑

---

## 🚀 快速开始

### 方式一：配置工具（首次使用推荐）

```bash
# 双击运行
配置工具.bat
```

按提示输入：
1. **API Key**（阿里云 DashScope）
2. **选择模型**（1-8 选项）

配置会自动保存，下次启动直接使用！

---

### 方式二：双击启动

```bash
# 双击运行
启动编程平台.bat
```

启动后在平台内输入：
```
/api sk-sp-cec7682a7c994e489716da0ad751ce1e
/model qwen3.5-plus
```

---

### 方式三：命令行启动

```bash
# 设置环境变量
setx AI_API_KEY "sk-sp-cec7682a7c994e489716da0ad751ce1e"
setx AI_MODEL "qwen3.5-plus"
setx AI_PROVIDER "dashscope"

# 启动平台
cd "C:\Users\Administrator\Desktop\火星文件扩展"
node mars-platform.js
```

## 📋 命令大全

### 基础命令

| 命令 | 说明 | 示例 |
|------|------|------|
| `/help` | 显示帮助文档 | `/help` |
| `/quit` | 退出程序 | `/quit` |
| `/clear` | 清空对话历史 | `/clear` |
| `/config` | 显示配置信息 | `/config` |
| `/status` | 显示系统状态 | `/status` |

### AI 配置命令

| 命令 | 说明 | 示例 |
|------|------|------|
| `/api <key>` | 设置 AI API Key | `/api sk-sp-xxx` |
| `/model <模型>` | 切换 AI 模型 | `/model qwen3.5-plus` |

### MCP 管理命令

| 命令 | 说明 | 示例 |
|------|------|------|
| `/mcp list` | 列出所有 MCP 服务器 | `/mcp list` |
| `/mcp add <名> <命令>` | 添加 MCP 服务器 | `/mcp add fs npx -y @mcp/fs` |
| `/mcp remove <名>` | 删除 MCP 服务器 | `/mcp remove fs` |
| `/mcp enable <名>` | 启用服务器 | `/mcp enable fs` |
| `/mcp disable <名>` | 禁用服务器 | `/mcp disable fs` |
| `/mcp test <名>` | 测试连接 | `/mcp test fs` |

## ⚙️ 配置说明

### 环境变量

| 变量 | 说明 | 默认值 |
|------|------|--------|
| `AI_API_KEY` | AI API 密钥 | - |
| `AI_PROVIDER` | API 提供商 | `dashscope` |
| `AI_MODEL` | 模型名称 | `qwen3.5-plus` |
| `AI_BASE_URL` | API 地址 | `https://coding.dashscope.aliyuncs.com/v1` |
| `FM_WORKSPACE` | 工作区路径 | 火星文件目录 |

### 获取 API Key

1. 访问阿里云 DashScope 控制台
2. 创建/获取 API Key
3. 在平台中输入 `/api <key>`

## ⚠️ 常见问题

### 问题 1：未配置 API Key

```
⚠️ 未配置 AI API Key
```

**解决：**
```
/api sk-sp-your-key
```

### 问题 2：火星文件 CLI 不存在

```
❌ 火星文件 CLI 不存在
```

**解决：**
```bash
cd 火星文件
npm install
npm run build
```

### 问题 3：模型切换失败

```
⚠️ 未知的模型 ID: xxx
```

**解决：** 使用 `/config` 查看支持的模型列表
